# disney-step-checker
# Disney STEP トレジャー収集チェッカー

Disney STEPのトレジャー（★2、★3）をカテゴリごとに管理できる、個人向けのチェックリストツールです。
スタンプラリーのラッキートレジャーを探すために、既出のトレジャーを把握する目的で作成しました。

## ✨ 特徴
- **カテゴリ別折りたたみ機能**: シリーズごとにスッキリ整理。
- **リアルタイム進捗**: 収集率がひと目でわかります。
- **プライバシー配慮**: チェックデータはサーバーに保存されず、あなたのブラウザ内（ローカルストレージ）のみで完結します。

## 🚀 使い方
公開URLにアクセスし、取得したトレジャーにチェックを入れてください。
「ホーム画面に追加」を行うことで、アプリのように起動できます。
